# Screenshots

This folder contains screenshots and images for the MediChat Pro README.

## Instructions to Add Screenshot

1. Save your screenshot as `app_interface.png` in this folder
2. Commit and push to GitHub:
   ```bash
   git add app_interface.png
   git commit -m "Add app interface screenshot"
   git push origin main
   ```

The screenshot will then display in the README.md file on GitHub automatically!
